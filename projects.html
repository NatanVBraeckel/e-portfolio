Eisenanalyse

Dit project was in opdracht van het vak Requirements Analysis. In groep kregen een opdracht van een klant die de administratie van zijn hockeyclub wou digitaliseren.

Alle use cases werden verdeeld over de groepsleden. Bij elke use case werd een beschrijving en een prototype gemaakt. In dit project heb ik geleerd om een project vanuit een ander standpunt te bekijken.

Space Info Station

Dit project was in opdracht van het vak Webdesign Advanced. In groep mochten we een thema kiezen. Voor dat thema moest elk teamlid 2 pagina's maken met behulp van Bootstrap componenten, Sass en JavaScript. Wij hadden gekozen voor een ruimte-thema.

Dit project verliep niet vlekkeloos. Er was 1 iemand in onze groep die zelden van zich liet horen, en dat zorgde voor veel vertragingen. Dat heeft er tot geleid dat ik knopen moest doorhakken en de leiding moest nemen, iets dat ik normaal niet doe. Ondanks de problemen vind ik het project zeker geslaagd en ben trots op de JavaScript die ik heb geschreven en dat ik het voortouw heb genomen.

Wat gaan we eten?

Dit project was in opdracht van Professional Skills. De opdracht was om een eigen idee uit te werken met behulp van een technologie die je nog niet kende. Ik had ervoor gekozen om met Vue te werken, een JavaScript framework met gelijkenissen aan Angular en React.

Het project geeft een overzicht van gerechten waar men in kan filteren, gerechten kan toevoegen en verwijderen.
