Aanpak

Ik had gekozen om met Vue.js te werken, een JavaScript framework zoals React en Angular. Ik heb hierover bijgeleerd via Open Classrooms, een website waar je gratis bij kan leren over verschillende onderwerpen.

Het project lost de veel voorkomende vraag 'Wat gaan we eten?' op, door de gebruiker een soort van kookboek te geven waarin men gerechten kan aanmaken, filteren, verwijderen of als favoriet kan aanduiden.

Conclusie

In dit project heb ik op mijn eentje een nieuwe technologie geleerd. Ik heb dus geleerd hoe ik zelfstandig op zoek kan gaan naar informatie. Ook ben ik wat meer uit de creatieve hoek moeten komen, met het opkomen van een thema.
