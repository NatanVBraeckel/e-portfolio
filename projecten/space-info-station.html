Space Info Station

Website

Achtergrond

Een website maken die bezoekers iets bijleert over een zelf gekozen onderwerp. Iedere student maakt 2 pagina's waarin een Bootstrap component geïmplementeerd is en een toevoegend stuk JavaScript aan vastgebonden is.

  • Opleingsonderdeel Webdesign Advanced
  • Tweede semester eerste opleidingsjaar
  • Groep van 3 studenten

Technologieën

  • HTML & CSS
  • Sass
  • Bootstrap classes
  • Bootstrap componenten
  • JavaScript

Aanpak

We hebben als groep een ruimtethema gekozen. Ik maakte een pagina over aardse planeten en een pagina over de gasreuzen. De Bootstrap componenten die ik heb gebruikt zijn progress bars en een carousel. Met JavaScript heb ik een veranderent kleurenschema en een quiz gemaakt.

Het project verliep niet vlekkeloos. Eén groepsgenoot bracht ons zelden op de hoogte waardoor beslissingen laat genomen werden. Uiteindelijk hebben ik en Semih, mijn andere groepsgenoot, besloten om zonder hem door te werken.

Ik ben niet echt het type om feedback te geven aan mensen als ze daar niet om vragen, ookal zou ze dat met hun huidige problemen helpen. Maar Semih had regelmatig moeite met zijn pagina's en de omzetting van Sass naar CSS, waardoor ik hem vaak kon helpen en dus hem steeds meer feedback durfde geven.

Conclusie

In dit project heb ik geleerd om met Sass, Bootstrap en JavaScript te werken. Ook heb ik bijgeleerd over de Github workflow.

In dit project heb ik ook geleerd hoe ik met een groepsgenoot moet werken die niet meewerkt. Ik heb bijgeleerd om de touwtjes zelf in handen te nemen en mijn medestudenten feedback te geven.
